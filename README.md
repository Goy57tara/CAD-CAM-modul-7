# 📘 SolidWorks Assembly – Materi Lengkap

Repository ini berisi materi lengkap mengenai **Assembly pada SolidWorks**, mulai dari konsep dasar hingga praktik penggunaan mate dalam menyusun komponen menjadi satu sistem.

---

# 🎯 Pengertian Assembly

Assembly adalah proses menggabungkan beberapa **part (komponen)** menjadi satu kesatuan produk yang dapat dianalisis, disimulasikan, dan divisualisasikan.

Contoh:
- Mesin
- Gear system
- Rangka mekanik
- Produk manufaktur

---

# ⚙️ Tujuan Assembly

- Mengetahui hubungan antar komponen
- Mensimulasikan pergerakan
- Mengecek tabrakan (interference)
- Menyusun produk secara virtual
- Validasi desain sebelum produksi

---

# 🧠 Konsep Dasar

## 🔹 Part vs Assembly
- **Part** → Komponen tunggal
- **Assembly** → Gabungan beberapa part

## 🔹 Sub-Assembly
Assembly yang menjadi bagian dari assembly yang lebih besar.

---

# 🔩 Mate (Kunci Utama Assembly)

Mate adalah fitur untuk menghubungkan antar part agar memiliki hubungan tertentu.

## 📌 Fungsi Mate
- Mengunci posisi
- Mengatur gerakan
- Membuat hubungan realistis

---

# 🔧 Jenis Mate

## 1. Standard Mate

- **Coincident** → Menempel
- **Parallel** → Sejajar
- **Perpendicular** → Tegak lurus
- **Distance** → Jarak tertentu
- **Concentric** → Satu sumbu

---

## 2. Advanced Mate

- **Limit Mate** → Membatasi gerakan
- **Angle Mate** → Mengatur sudut
- **Symmetric** → Simetri

---

# 🛠️ Langkah Membuat Assembly

1. Buat file Assembly
2. Insert part pertama
3. Set sebagai **Fix**
4. Tambahkan part lain
5. Gunakan Mate untuk menghubungkan
6. Sesuaikan posisi dan gerakan

---

# 🔄 Fix vs Float

- **Fix** → Part tidak bisa bergerak
- **Float** → Part bisa bergerak

📌 Catatan:
Part pertama biasanya di-Fix sebagai referensi.

---

# ⚠️ Interference Detection

Digunakan untuk:
- Mendeteksi tabrakan antar part
- Menghindari kesalahan desain

---

# 🎯 Motion & Simulation (Dasar)

Assembly memungkinkan:
- Simulasi gerakan
- Analisis hubungan antar komponen

Contoh:
- Gear berputar
- Lengan robot bergerak

---

# 🧩 Best Practice

✔ Gunakan mate secukupnya  
✔ Hindari over-defined  
✔ Gunakan sub-assembly untuk proyek besar  
✔ Gunakan nama part yang jelas  
✔ Gunakan reference geometry jika diperlukan  

---

# ❌ Kesalahan Umum

- Terlalu banyak mate
- Tidak menggunakan fix dengan benar
- Part tidak sejajar
- Tidak cek interference

---

# 📌 Workflow Ideal

1. Buat Part (Modeling)
2. Masuk ke Assembly
3. Insert Part
4. Fix Part Utama
5. Gunakan Mate
6. Cek Interference
7. Simulasi

---

# 🎯 Kesimpulan

Assembly adalah tahap penting dalam CAD karena:
- Menggabungkan part menjadi sistem
- Membantu simulasi dan analisis
- Digunakan dalam dunia industri

Mate adalah kunci utama dalam assembly.
